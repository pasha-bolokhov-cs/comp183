CRAPS Instruction Set

Alphabetical listing

Key In three-operand instructions, dest = destination, src1 = source register 1, src2 = source register 2
In single-operand instructions op = operand
In flow of control instructions (branch, call) target = 16-bit immediate 
operand, src = source register
In load/store (and lea) instructions address = 16-bit immediate operand, 
src = source register
Flags O overflow
S sign
Z zero
C carry
Name Add with carry
Mnemonic ADC
Function dest := src1 + src2 + carry flag
Flags OSZC set
Note A 16-bit immediate value may be used for src2
Name Add
Mnemonic ADD
Function dest := src1 + src2
Flags OSZC set
Note A 16-bit immediate value may be used for src2
Name Logical AND
Mnemonic AND
Function dest := src1 AND src2 
(bit n of dest = 1 iff bit n of src1 and bit n of src2 are both 1, 
otherwise bit n of dest = 0)
Flags OC set to zero, SZ set
Note A 16-bit immediate value may be used for src2
Name Branch if condition is met
Mnemonic Bcc
Function If appropriate condition is true pc := target + src 
(Condition flags will have been set by a previous instruction)
Flags None
Examples  
BC branch if carry flag on
BNC branch if carry flag off
BO branch if overflow flag on
BNO branch if overflow flag off
BS branch if sign flag on
BNS branch if sign flag off
BZ branch if zero flag on
BNZ branch if zero flag off
BE branch if equal
BA branch if above (unsigned)
BAE branch if above or equal (unsigned)
BB branch if below (unsigned)
BBE branch if below or equal (unsigned)
BG branch if greater (signed)
BGE branch if greater or equal (signed)
BL branch if less (signed)
BLE branch if less or equal (signed)
Note The src register may be omitted (r0 is assumed)
Name Branch (unconditional)
Mnemonic BR
Function pc := target + src
Flags None
Note The src register may be omitted (r0 is assumed)
Name Call procedure
Mnemonic CALL
Function pc pushed onto stack, pc := op + src
Flags None
Note The src register may be omitted (r0 is assumed)
Name Compare
Mnemonic CMP
Function Compare src1 to src2, set flags
Flags OSZC
Note A 16-bit immediate value may be used for src2
Name Decrement by 1
Mnemonic DEC
Function dest := op - 1
Flags OSZ
Name Divide (unsigned)
Mnemonic DIV
Function dest := src1 / src2 (integer - i.e truncated - division) 
dest+1 := src1 modulo src2 (i.e. the remainder)
Flags C and O are cleared, S and Z are undefined.
Notes The results occupy two registers; dest+1 signifies the next numbered register. 
A 16-bit immediate value may be used for src2
Name Halt
Mnemonic HALT
Function The CPU stops 
Note: use of this instruction is not recommended by user 
programs and may cause an error in future versions
Flags OSZ
Name Increment by 1
Mnemonic INC
Function dest := op + 1
Flags OSZ
Name Load effective address offset
Mnemonic LEA
Function dest := address + src
Flags None
Note The src register may be omitted (r0 is assumed)
Name Load Byte High
Mnemonic LODBH
Function dest (bits 0-7) := M[address + src] 
dest (bits 8-15) unchanged
Flags None
Note The src register may be omitted (r0 is assumed)
Name Load Byte Low
Mnemonic LODBL
Function dest (bits 8-15) := M[address + src] 
dest (bits 0-7) unchanged
Flags None
Note The src register may be omitted (r0 is assumed)
Name Load Byte Signed
Mnemonic LODBS
Function dest (bits 8-15) := M[address + src] 
dest (bits 0-7) all set to copy of bit 8 
(i.e. the sign bit of the original byte)
Flags None
Note The src register may be omitted (r0 is assumed)
Name Load Byte Unsigned
Mnemonic LODBU
Function dest (bits 8-15) := M[address + src] 
dest (bits 0-7) set to zero
Flags None
Note The src register may be omitted (r0 is assumed)
Name Load Word
Mnemonic LODW
Function dest (bits 0-7) := M[address + src] 
dest (bits 8-15) := M[address + src + 1]
Flags None
Note The src register may be omitted (r0 is assumed)
Name Move data
Mnemonic MOV
Function dest := op
Flags OC set to zero, SZ set from operand
Name Multiply (unsigned)
Mnemonic MUL
Function dest := src1 * src2 (bits 0-15) 
dest+1 := src1 * src2 (bits 16-31)
Flags If the result will fit into 16 bits (i.e. dest = 0) then C and O are cleared, 
otherwise they are set. S and Z are undefined.
Notes The result occupies two registers; dest+1 signifies the next numbered register. 
A 16-bit immediate value may be used for src2
Name Two's complement negation
Mnemonic NEG
Function dest := -op
Flags OSZC
Name No operation
Mnemonic NOP
Function pc := pc + 4 (address of next instruction)
Flags None
Name One's complement negation
Mnemonic NOT
Function If bit n of op is 1 bit n of dest := 0, and vice-versa
Flags None
Name Logical inclusive OR
Mnemonic OR
Function dest := src1 OR src2 
(bit n of dest = 0 iff bit n of src1 and bit n of src2 are both 0, 
otherwise bit n of dest = 1)
Flags OC set to zero, SZ set
Name Pop stack to register
Mnemonic POP
Function op (bits 0-7) := M[sp] 
op (bits 8-15) := M[sp+1] 
sp := sp - 2
Flags None
Name Push register onto stack
Mnemonic PUSH
Function sp := sp + 2 
M[sp] := op (bits 0-7) 
M[sp+1] := op (bits 8-15)
Flags None
Name Return
Mnemonic RET
Function pc is popped from the stack
Flags None
Name Shift arithmetic right
Mnemonic SAR
Function src1 is shifted right src2 bits and stored in dest 
vacated left bits are filled with a copy of bit 0 of src1 
(sign extension)
Flags SZ set, C set off, O unchanged
Name Subtract with Borrow
Mnemonic SBB
Function dest := src1 - src2 - carry flag
Flags OSZC set
Name Shift left
Mnemonic SHL
Function src1 is shifted left src2 bits and stored in dest 
vacated right bits are filled with zero
Flags SZC set, O unchanged
Name Shift right
Mnemonic SHR
Function src1 is shifted right src2 bits and stored in dest 
vacated left bits are filled with zero
Flags SZ set, C set off, O unchanged
Name Store Byte High
Mnemonic STOBH
Function M[address + src1] := src2 (bits 0-7)
Flags None
Note The src1 register may be omitted (r0 is assumed)
Name Store Byte Low
Mnemonic STOBL
Function M[address + src1] := src2 (bits 8-15)
Flags None
Note The src1 register may be omitted (r0 is assumed)
Name Store Word
Mnemonic STOW
Function M[address + src1] := src2 (bits 0-7) 
M[address + src1 + 1] := src2 (bits 8-15)
Flags None
Name Subtract
Mnemonic SUB
Function dest := src1 - src2
Flags OSZC set
Name System call
Mnemonic SYS
Function The CRAPS BIOS (CRAB) is invoked and passed the immediate value
Flags None
Name Logical test
Mnemonic TEST
Function r0 := src1 and src2
Flags OC set to zero, SZ set
Name Logical exclusive or
Mnemonic XOR
Function dest := src1 xor src2 
(bit n of dest = 1 iff bit n of src1 and bit n of src2 are equal, 
otherwise bit n of dest = 0)
Flags OC set to zero, SZ set
