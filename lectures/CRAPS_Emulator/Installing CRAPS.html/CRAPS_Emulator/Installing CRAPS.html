Installing CRAPS

CRAPS currently only runs on GNU/Linux systems (32- or 64-bit); it requires python 2.x (where x >=4), and not python 3.x.

If you wish to run it on your own system, download this gzipped tar file, unpack it in some convenient place and then read the REAME.txt file for further instructions.

There are a couple of reasons why CRAPS might not run - or might produce strange messages - on your system:

  1. Your system is 32-bit (the binaries in the linked package are 64-bit)
  2. You are running a different version of python (you will get messages about library versions, although it will still - probably - run)

In both these cases it will be necessary to rebuild the system, using the supplied Makefile. See the README.txt file for more details.
