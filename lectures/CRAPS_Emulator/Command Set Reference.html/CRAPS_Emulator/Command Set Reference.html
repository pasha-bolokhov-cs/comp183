CRAPS commands

Alphabetical listing

chasm

The chasm program is the assembler for the CRAPS system.

Usage:

    chasm [ -o imagename] [ -l ] [-t tabsize] program

where:

-o imagename pathname of image (machine code) file
-l produce a listing file
-t tabsize specify tabstop size. Horizontal Tab (HT) characters encountered in the source file will be expanded to the next tab stop column. The default is every 8 columns.
program path of source file. If no extension .csm is assumed. If -o omitted, image file name is program file name with extension replaced by .cig If -l specified, listing file name is the program file name with extension replaced by .lst

craps

The craps program is the main user interface to the CRAPS system. It contains the debugger (see manual) and chasm programs can be edited, assembled and loaded into the virtual machine's memory from the craps command line.

Usage:

    craps [-l imagefile] [-r imagefile] [-d] [imagefile]

where:

-l imagefile load imagefile (the .cig extensions is assumed) into the CRAPS memory.
-r imagefile load and run imagefile. If the program terminates abnormally, a termiantion status will be printed. The debugger loop is not entered
-d enter the main debug loop after the programs stops, even if it terminates normally (i.e. via the SYS EXIT call)
imagefile same as -r imagefile

The -l and -r options may not be combined.
