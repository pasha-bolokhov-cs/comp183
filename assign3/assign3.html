Instructions

Lab 3

For demonstration during week 5

15 Marks

(Work in pairs)

Topics

  • Procedures
  • Indexed addressing

Assignment

Write a program which will:

  1. Prompt the user for a string
  2. Read the string from the keyboad
  3. Tell the user whether the string is a palindrome

and keep doing the above until the user types just Enter.

A palindrome is a sentence which is spelled the same forwards or backwards. Normally this involves ignoring punctuation, spacing and case - for example the two most famous palindromes are probably:

Madam, I'm Adam

and

Able was I, ere I saw Elba

You should have at least one subroutine (procedure) in your program and will lose marks if you don't.

Hints

Here you will find an image file of a working lab 3. (Yes, I have given you the answer -ish).

You can run this image file (the usual way) in order to see what the program is meant to do.

You can find more hints here, and the masochistic among you can see what you'd have been up against using assembler for the Intel chips (MASM) here.

Hand in

A single copy of your team's source code. (Provided I get the Dropbox configured for that correctly.)
