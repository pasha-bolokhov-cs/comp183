Lab 3 hints

This lab builds on what you did in lab 2, but where that one started at the end of the string and worked backwards, the best way to detect palindromes is to start at both ends and work inwards. 

Here is a python palindrome detector:

def is_pal (str):
    forward = 0                 # beginning
    backward = len (str) - 1    # end
    while forward < backward:
        if str[forward] != str[backward]:
            return False
        forward += 1            # next char
        backward -= 1           # previous char

    return True                 # passed all tests, must be palindrome

if __name__ == '__main__':
    instr = raw_input ("Enter a string ")
    while instr != "":          # until Enter alone is pressed
        print instr,
        if is_pal (instr):
            print "is",
        else:
            print "is NOT",
        print "a palindrome"
        instr = raw_input ("Enter a string ")

You already have the outer loop (while the input is not empty) in lab 2. Inside that loop you need to 

  • move the offset of the first input char to some register
  • move the offset of the last input char to some (other) register
  • while these two are not crossed:
    • load the 'forward' character to some third register
    • load the backward' character to some fourth register
    • check to see if the two characters loaded are the same
    • if not you don't have a palindrome
    • if they are the same keep going

Note that you must included a subroutine in your code, which means you must have a CALL instruction somewhere (and a RETurn to go back).

The subroutine need not be complicated, I'd suggest maybe one to print the yes/no result, but it is entirely up to you.
